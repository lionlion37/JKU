This course covered the following topics:

    Reading-in of datasets, visualize
    Using Deep Learning frameworks like Pytorch
    Playing around with the first Artificial Neural Networks
    Vanishing gradient problem, visualization and solutions
    Testing and playing around with various different Deep Learning architectures, e.g. Convolutional Neural Networks, LSTM, Autoencoders, Generative Adversarial Networks, ...
    Looking at simple Reinforcement Learning problems
    Looking at latest Reinforcement Learning problems and applying Deep Learning techniques
