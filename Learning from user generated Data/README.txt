This course covered the following topics:

    Sources of user-generated data
    mining and analysis of content from web and social media
    mining of user behaviour and feedback (explicit and implicit)
    recommender systems
    personalisation
    content-based filtering
    collaborative filtering
    context-awareness
    hybridization
    bias and fairness 

At the end of the course, there was a challenge to create a recommender system from scratch (see folder "Challenge").
